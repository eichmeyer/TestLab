Die ist das Build- Script meiner alte Test- Umgebung. Dies Installierte mir eine komplette Test- Umgebung mit folgenden Eigenschaften:

-	2x AD Controller inkl. DNS/DHCP
-	S2D Storage- Cluster optional inkl. Hyper-V 
-	Optional S2D Cluster basierend auf Nano Server inkl. Hyper-V
-	Alle VMs basieren auf Basis VHDs mit Hilfe von differenzierenden VHDX
-	Die Konfiguration der VMs Erfolg mittels PowerShell Direct
-	Es wird Hyper-V mit genügend RAM benötigt

Die Konfiguration erfolgt über die Parameter im Script.
Benötigt wird eine Evaluationslizenz von Windows Server 2016.
